# IPv6 CoAP Example

Этот пример демонстрирует работу с IPv6 адресами в библиотеке coalago.

## Возможности

- Сервер с поддержкой IPv6 адресов
- Клиент для тестирования IPv6 соединений
- Несколько тестовых эндпоинтов для демонстрации функциональности

## Использование

### Запуск сервера

```bash
go run main.go --mode server
```

Сервер будет слушать на `[::1]:5683` (IPv6 localhost).

### Запуск клиента

```bash
go run main.go --mode client
```

Клиент подключится к серверу и выполнит несколько тестов.

### Дополнительные параметры

```bash
# Указать другой IPv6 адрес
go run main.go --mode server --addr [2001:db8::1] --port 8080

# Подключиться к удаленному серверу
go run main.go --mode client --addr [2001:db8::1] --port 8080
```

## Доступные эндпоинты

- `GET /info` - информация о сервере
- `POST /echo` - эхо тест (возвращает отправленные данные)
- `GET /time` - текущее время сервера
- `GET /ipv6` - подтверждение IPv6 соединения

## Примеры IPv6 адресов

- `[::1]` - IPv6 localhost (аналог 127.0.0.1)
- `[::]` - все IPv6 интерфейсы
- `[2001:db8::1]` - пример глобального IPv6 адреса
- `[fe80::1%lo0]` - link-local адрес с зоной (на Unix системах)
